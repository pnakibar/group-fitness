# group-fitness
## Run
### Pre-requisites
- Android Emulator
- React-native
- [Setting up the environment](https://facebook.github.io/react-native/docs/getting-started.html#content)

### Android
#### Start the Android Emulator
```bash
emulator -list-avds
emulator -avd 'avd name'
```
#### Serve the content
```bash
react-native run-android
npm start
```
